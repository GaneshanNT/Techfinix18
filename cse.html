Crypsode

×

Description

Debugging is twice as hard as writing the code. The event is to test the deep knowledge of the programming skills. Programming skills enable you to take these tasks into your own hands and forge the solution yourself
  • Round 1:  One mark (20 multiple choice questions).
  • Round 2: A program will be given and you have to write the output for the given program.
  • Round 3: Output will be given and you have to write the code for the output.

    • Rules:

    • Each team can have a maximum of two members.
    • Third party libraries will not be expected and will not fetch any points.
    • Only C and C++ are allowed.
    • Disassembler tools will not be provided for the final round. Participants have to rely on their analytic skills to reverse‐engineer the source code of the given executable file.
    • Results of all the rounds are not subject to discussion and judge’s decision is final.
